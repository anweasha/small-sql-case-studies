## WebDevSimplified - Bands & Albums

This repository contains my solutions to the SQL exercises curated by [WebDevSimplified](https://github.com/WebDevSimplified/Learn-SQL).

### Table of Contents
- [schema.sql](schema.sql) contains the SQL code to create and define the initial tables.
- [data.sql](data.sql) includes the statements to populate the tables with values.
- [Exercises.md](Exercises.md) lists all the questions along with my solutions and their outputs for each.

**Note:** These exercises were completed using MySQL on [DB Fiddle](https://www.db-fiddle.com/).
